# Kota Baru Indramayu - Single Page Website

## Project Overview
Single page website for Kota Baru Indramayu project by PT Laksana Desa Manajemen, featuring bilingual support (Indonesian and English).

## Features
- **Bilingual Support**: Toggle between Indonesian (ID) and English (EN)
- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional design based on business template
- **Single Page Navigation**: Smooth scrolling between sections
- **Language Persistence**: Language toggle functionality

## Project Structure
```
singlepage/
├── index.html          # Main HTML file with all content
├── css/                # Stylesheets
│   ├── vendors.min.css # Third-party CSS libraries
│   ├── icon.min.css    # Icon fonts
│   ├── style.css       # Main styles (from template)
│   └── responsive.css  # Responsive design rules
├── js/                 # JavaScript files
│   ├── jquery.js       # jQuery library
│   ├── vendors.min.js  # Third-party JS libraries
│   └── main.js         # Main JavaScript functionality
└── images/             # Image assets (currently placeholder)
```

## Language Support
The website supports two languages:
- **Indonesian (ID)** - Default language
- **English (EN)** - Alternative language

### How to Use Language Toggle
1. Click on "ID" or "EN" buttons in the header
2. All content will automatically switch to the selected language
3. The active language button will be highlighted

## Technical Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required (can run locally)
- JavaScript enabled for language switching functionality

## Browser Compatibility
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Customization
To modify content or styling:

1. **Content Changes**: Edit the `languageContent` object in the JavaScript section of `index.html`
2. **Styling Changes**: Modify the CSS in the `<style>` section or in the CSS files
3. **Adding Sections**: Add new HTML sections following the existing pattern
4. **Images**: Replace placeholder images with actual project images in the `images/` folder

## Development Notes
- The website uses Bootstrap 5 classes for responsive design
- Font Awesome icons are used for visual elements
- Custom JavaScript handles language switching
- All content is embedded in a single HTML file for easy deployment

## Deployment
Simply upload all files to any web server or hosting provider. No database or server-side processing required.
